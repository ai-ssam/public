# Helper Monitor - Project Plan & Roadmap

이 문서는 프로젝트의 개발 로드맵과 버전별 변경 사항을 기록합니다.

## 📅 Version History

### [v1.0.0] - 초기 릴리즈 (Completed)
**목표**: 14가지 필수 모니터 테스트 기능을 갖춘 웹 애플리케이션 구현.
- **기간**: 2026-01-31
- **주요 기능**:
    - 14종 테스트 (불량화소, 명암비, 가독성, 응답속도, 주사율 등)
    - 다크 모드 UI
    - F11 전체 화면 지원
    - 모듈화된 JS 구조

### [v1.1.0] - 수익화 및 최적화 (Completed)
**목표**: 구글 애드센스 연동을 통한 수익 모델 구축 및 UI 개선.
- **구현 기능**:
    - 구글 애드센스 스크립트 연동 (`<head>`)
    - 상단(Header) 배너 광고 영역 추가
    - 하단(Footer) 디스플레이 광고 영역 추가
    - 광고 로딩 시 레이아웃 밀림(CLS) 방지 스타일링

### [v1.2.0] - 글로벌 확장 및 수익화 강화 (Completed)
**목표**: 다국어 지원 및 광고 영역 확장.
- **구현 기능**:
    - **다국어 지원 (i18n)**: 11개국 언어 지원 (영어, 스페인어, 프랑스어, 독일어, 중국어, 일본어, 한국어, 터키어, 포르투갈어, 이탈리아어, 러시아어).
    - **심층 번역**: 14개 테스트 항목 전체에 대한 상세 가이드 및 경고 메시지 번역 적용.
    - **Flag Icons UI**: `flag-icons` 라이브러리를 활용한 고품질 국기 아이콘 및 원형 버튼 디자인 적용.
    - **사이드 광고**: 데스크탑 화면 좌우측에 스카이스크래퍼(160x600) 광고 영역 추가.
    - **레이아웃 변경**: `[Left Ad] - [Content] - [Right Ad]` 3단 반응형 구조 적용 (모바일 대응).

### [v1.3.0] - 사용자 경험 및 기능 확장 (Planned)
**목표**: 사용자 설정 저장 및 모바일 경험 강화.
- **예정 기능**:
    - **LocalStorage 저장**: 언어 설정 및 마지막 테스트 상태 저장.
    - **PWA (Progressive Web App)**: 앱 설치 지원.
    - **터치 제스처**: 모바일 환경에서의 스와이프 테스트 지원.
    - **단축키 안내**: 테스트 시 단축키 툴팁 제공.

## 🚀 완료된 주요 작업 (v1.2.0)

### 1. 다국어 시스템 고도화 (`js/i18n.js`)
- `t(key)` 메서드를 통한 통합 번역 처리.
- 테스트 스크립트(`js/tests/*.js`) 내 하드코딩 텍스트 전면 교체.
- `flag-icons` CSS 라이브러리 도입으로 시각적 퀄리티 향상.

### 2. 반응형 광고 레이아웃 (`index.html`, `style.css`)
- **Desktop**: 중앙 1000px 컨텐츠 + 좌우 고정 광고 영역.
- **Mobile**: 사이드 광고 자동 숨김 및 컨텐츠 100% 폭 사용.
